# Fitur Umroh App

Aplikasi Fitur Umroh: Jadwal Sholat (bimasislam.kemenag.go.id) dan Arah Kiblat, dibuat menggunakan Flutter versi 2.5.3

<br>

## Fitur

- Halaman Login dan Daftar
- Halaman Jadwal Sholat
- Halaman Arah Kiblat
