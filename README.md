# Fitur Umroh App

Aplikasi Fitur Umroh seperti Jadwal Sholat dan Arah Kiblat, dibuat menggunakan Flutter versi 2.5.3


## Fitur

- Halaman Login dan Daftar
- Halaman Jadwal Sholat
- Halaman Arah Kiblat
